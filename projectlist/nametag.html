Name Tag 3D model

This page will go through how to model keychain tag using Fusion 360.

1st Sketch

From the browser, open up the origin tab, select the xy plane, right-click and create new sketch.


Draw a horizontal slot from the origin that is 70mm long end-to-end and 30mm wide.


At the end of the slot, draw a 4mm diameter circle and use the offset tool to create a larger circle, 1.5mm offset from the original circle.


Extrude

With the extrude tool, select the whole 1st sketch except for the dia 4mm hole, and extrude it by 4mm.


Shelling

With the shelling tool, select the top face, set direction as "inside", set thickness as 1.5mm and click ok.


Sketch Name

Create a new sketch on the bottom face of the shell.


Draw 2 centerlines across the tag.


Under "create", select "Text", and create a text box. Enter your name, set size to preferred size and set alignment to center and middle. Click ok when done.

Contrain the text box to the centerlines with symetrical constrain to fixe it to the middle of the tag.

Finish sketch when done.


Extrude Name

Select the name to extrude and extrude by 2.5mm, or to the top face of the tag.


You should now have a completed model of a name tag. This is the end of the documentation
