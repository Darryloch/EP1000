Electronics Design

This si how I design PCBs in my SP EP1001 module.
I used Autodesk Fusion 360 and GIMP softwares for this.

Libraries

Electronics design libraries that I used can be found here

Fusion 360

Create new electronics design in fusion and create new schematic diagram. Add the downloaded libraries from the library manager or if unable to do so, drag and drop into the libraries folder of Fusion 360 in the computer file explorer. Make sure the libraries are activated in the library manager.

Next, place all the needed components using the "Place components" tool. Search for the components by entering key-words or components code in the search bar. Arrange all the components and connect them with the "Net" tool.

Once the circuit is completed, switch to PCB document to bring you to a space with all the components and their footprints. Before anything else, click on "DRC" under the "Rules DRC/ERC" tab. Input the line width and clearance as 18 mils and apply.
Arrange all the components neatly according to liking and how they can be most easily connected, then connect all the components with the routing tool. For convenience, I used the auto-router tool to fast connect everything first, I then improved the connections from there. Use the error checker tool to ensure you ahve everything properly connected

Once everything is connected, it's time to create the PCB outline. Select the "Board outline" tool, then select the "milling" layer, then draw out a desired board shape around the circuit. Ensure that the outline is fully closed.

Once the PCB design is fully completed, it's time to export it into images. Since Fusion 360 does not have a function for that, you have to input a command line. The command is:
export image monochrome <drive:\path\filename.png> <resolution>
e.g. export image monochrome c:\ep10001\updi_traces.png 1000

Under display layers, select display layers, disable everything except the top and pads layer and export image. this will be the PCB traces.
Next, disable everything except for the milling layer and export image again, this will be the PCB outline.

GIMP

Open the images in GIMP, crop the design image out to an appropriate size and fill in the inside spaces to white and leave the outside spaces as black.
The white spaces are the parts left untouched during milling and the the outline of the white spaces in the black spaces are cut during PCB milling.

Now you have the images ready to carry out production.
